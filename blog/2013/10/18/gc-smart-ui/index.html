概述

目标和对象

  1. 目标：在不改变C++底层框架的基础上，程序能做到满足需求的灵活修改。

  2. 对象：游戏逻辑层。UI,SKILL,CONTROL,etc…

行为树实现

Task = {
    update = function(args) end,
    on_enter = function(args) end,
    on_exit = function(args) end,
}
BehaviorNode = {
    nodes = {},
    status = 0,
    task = nil,
    tick = function(args)
        return task:update(args)
    end,
}
BehaviorTree = {
    object = nil,
    root = nil,
    tick = function(args)
        if root then
            root:tick(args)
        end
    end,
}
